# ADT(추상자료형)

컴퓨터 과학에서

Abstract Data Type

자료들과 그 자료들에 대한 연산들을 명기한 것

구현 방법을 명시하고 있지 않다

각 연산의 시간복잡도 명기 X

추상적 자료형은 인터페이스 와 구현을 분리하여 추상화 계층을 둔 것 

추상적 자료 구조

자료에 대한 일련의 연산이 정의되며, 각각의 연산에 대한 연산 복잡도가 정의된 가상의 자료 저장 공간

자료의 구체적인 구현 방식과는 관련이 없다.
